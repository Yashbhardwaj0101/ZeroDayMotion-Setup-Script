# ZeroDayMotion Auto-Setup Script 🚀

Yeh script **ZeroDayMotion** channel ke liye design ki gayi hai. Iska main maqsad aapke Linux system ko kuch hi seconds mein ek "Hacker/Developer" station mein convert karna hai.

## Features ✨
- ✅ System update aur upgrade ka automatic handling.
- ✅ Essential CLI tools (btop, fastfetch, fzf, etc.) ki automatic installation.
- ✅ Flatpak support aur productivity apps (VS Code, Discord, Spotify) ka setup.
- ✅ Oh My Zsh ka automatic installation for pro terminal looks.

## Kaise Use Karein? 🛠️

1. **Terminal kholein** aur is repository ko clone karein (ya file download karein):
```bash
   git clone [https://github.com/YOUR_USERNAME/ZeroDayMotion-Setup-Script.git](https://github.com/YOUR_USERNAME/ZeroDayMotion-Setup-Script.git)
   cd ZeroDayMotion-Setup-Script
2. Script ko Permissions dein:

Bash
   chmod +x setup.sh
3. Script run karein:

Bash
   ./setup.sh
Disclaimer ⚠️
Yeh script system-level changes karti hai. Ise run karne se pehle apni zaroori files ka backup zaroor le lein.
